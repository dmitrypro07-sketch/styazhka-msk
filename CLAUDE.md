# CLAUDE.md — Стяжка.МСК (Радомир)

## Проект
Лендинг для **Радомира** — полусухая стяжка пола в Москве и МО.
Один HTML-файл, всё inline (HTML + CSS + JS).

> **Примечание:** папка называется `nalivnye_poly_msk`, но сайт про **полусухую стяжку**, не наливные полы. Услуга — стяжка.

## Деплой
- **GitHub:** https://github.com/dmitrypro07-sketch/styazhka-msk
- **Vercel:** проект `nalivnye_poly_msk`
- **Автодеплой:** `git push origin main` → Vercel

## Стек
- HTML5 / CSS3 / Vanilla JS (один файл)
- Google Fonts: Bebas Neue (заголовки), DM Sans (текст), DM Mono (теги/метки)
- Нет npm, нет сборщика — просто открываем index.html

## Разделение палитр между проектами
- **Радомир (Москва)** — антрацит + **оранжевый**
- **Никита/ПолСтеныч (СПб)** — шоколад + **золото**
Не путать. Если меняли по ошибке — вернуть обратно.

## Правила
- Не менять цветовую схему без явной просьбы
- Не подключать внешние зависимости (npm и пр.)
- Калькулятор — только чистый JS
- ИИ-чат делает fetch на `/api/chat` (POST, body: {message, history, system})
- При отсутствии сервера — автоматически срабатывает fallback (getFallbackAnswer)
- Услуга — полусухая стяжка пола (не наливные полы, несмотря на имя папки)
