# CLAUDE.md — Наливные полы Москва

## Проект
Лендинг для компании по устройству наливных полов в Москве и МО.
Один HTML-файл, всё inline (HTML + CSS + JS).

## Стек
- HTML5 / CSS3 / Vanilla JS
- Google Fonts: Bebas Neue (заголовки), DM Sans (текст), DM Mono (теги/метки)
- Нет npm, нет сборщика — просто открываем index.html

## Структура файлов
```
nalivnye_poly_msk/
├── index.html     ← вся страница
├── project.md     ← документация
├── PLAN.md        ← чек-лист задач
└── CLAUDE.md      ← этот файл
```

## CSS-переменные (цвета)
```css
--bg: #0d0d1a        /* основной фон */
--bg2: #12121f       /* фон секций */
--bg3: #1a1a2e       /* карточки */
--accent: #4361ee    /* синий акцент */
--accent2: #7b8cff   /* светлее акцент */
--white: #f0f0f8     /* текст */
--muted: #8888aa     /* второстепенный текст */
```

## Правила
- Не менять цветовую схему без явной просьбы
- Не подключать внешние зависимости (npm и пр.)
- Калькулятор — только чистый JS
- ИИ-чат делает fetch на /api/chat (POST, body: {message, history, system})
- При отсутствии сервера — автоматически срабатывает fallback (getFallbackAnswer)

## Команды запуска
```
# Просто открыть в браузере:
start index.html

# Или через live-server:
npx live-server
```
