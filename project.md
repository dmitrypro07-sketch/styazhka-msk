# Наливные полы Москва — Лендинг

## Описание
Одностраничный лендинг для компании по устройству наливных полов в Москве и Московской области. Целевая аудитория — частные лица, ИП и компании, ищущие подрядчика по наливным полам.

## Файлы
| Файл | Назначение |
|------|------------|
| `index.html` | Главная страница — весь HTML, CSS, JS в одном файле |
| `project.md` | Документация проекта |
| `PLAN.md` | Чек-лист задач |
| `CLAUDE.md` | Правила работы с проектом |

## Технический стек
- Чистый HTML5 + CSS3 + Vanilla JS
- Google Fonts: Bebas Neue + DM Sans + DM Mono
- Без внешних фреймворков и зависимостей
- ИИ-чат делает fetch на `/api/chat` (нужен бэкенд)

## Секции лендинга
1. Hero — заголовок, статистика, CTA
2. Преимущества — 4 карточки
3. Услуги — 6 видов полов с ценами
4. Калькулятор — интерактивный расчёт стоимости
5. Портфолио — 6 объектов
6. Отзывы — 3 отзыва
7. Контакты — форма заявки
8. ИИ-помощник — чат-виджет (floating)

## Запуск
Открыть `index.html` в браузере. Для работы ИИ-чата нужен сервер с эндпоинтом `/api/chat`.
