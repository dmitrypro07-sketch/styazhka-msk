# Стяжка пола Москва — Лендинг Радомира

## Описание
Одностраничный лендинг для **Радомира** — мастера по механизированной полусухой стяжке пола в Москве и Московской области. Целевая аудитория — частные лица и юр.лица, ищущие подрядчика по стяжке.

> **Примечание:** папка проекта исторически называется `nalivnye_poly_msk`, но по факту сайт про **полусухую стяжку**. Услуга — стяжка, не наливные полы.

## Деплой
- **GitHub:** https://github.com/dmitrypro07-sketch/styazhka-msk
- **Vercel проект:** `nalivnye_poly_msk` (у `pdimaz07-4626s-projects`)
- **Автодеплой:** `git push origin main` → Vercel

## Технический стек
- Чистый HTML5 + CSS3 + Vanilla JS
- Google Fonts: Bebas Neue + DM Sans + DM Mono
- Без внешних фреймворков и зависимостей
- Деплой: Vercel + GitHub (репо `styazhka-msk`)
- ИИ-чат делает fetch на `/api/chat` (нужен бэкенд)

## Бренд и палитра
- **Бренд:** СТЯЖКА.МСК / Радомир
- **Регион:** Москва + МО
- **Палитра:** антрацит + оранжевый (свежая — поменяли с золота на оранж)
  ```
  --bg       #0f0f0f    антрацит (фон)
  --bg2      #141210    фон секций
  --bg3      #1a1714    карточки
  --accent   #ff6b1a    строительный оранжевый
  --accent2  #ff8c42    светлый оранжевый (hover)
  --white    #f5f0e8    тёплый молочный
  --muted    #8a8070    приглушённый текст
  ```

## Секции лендинга
1. Hero — заголовок, статистика, CTA
2. Преимущества — 4 карточки
3. Услуги — 6 видов полов с ценами
4. Калькулятор — интерактивный расчёт стоимости
5. Портфолио — 6 объектов
6. Отзывы — 3 отзыва
7. Контакты — форма заявки
8. ИИ-помощник — чат-виджет (floating)
9. Social-dock — Telegram / MAX / VK

## Запуск

```bash
start index.html                # локально

```
